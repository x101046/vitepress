Vue3常用 ​

基础库 ​

  • Vite -- 下一代前端开发与构建工具。

  • Pinia -- Pinia是一个更简洁的状态管理工具，Vue 2 和 Vue 3都适用。

  • Vue Router -- Vue.js 的官方路由

  • Axios -- Axios 是一个基于 promise 的网络请求库，可以用于浏览器和 node.js

  • Vue I18n -- Vue.js 的国际化插件

  • Vue DevTools -- Vue开发者工具

  • VueUse -- VueUse是基于Composition API 的实用函数集合，中文文档一般要低一个版本。

  • VueRequest -- 一个 Vue 请求库

  • WebTracing -- 一个基于 JavaScript 的埋点SDK

  • unplugin-vue-components -- 按需自动导入 Vue 组件

  • unplugin-auto-import -- 按需自动导入api

UI库 ​

  • Naïve UI -- 一个 Vue 3 组件库，比较完整，主题可调，使用 TypeScript，快。

  • Element Plus -- 基于 Vue 3，面向设计师和开发者的组件库。

  • DataV -- Vue 大屏数据展示组件库。

Css && 动画 ​

  • animate.css 最有名的动画效果库
  • anime.js 一个轻量级的JavaScript动画库，它使用CSS属性和SVG，可以创建平滑的CSS和SVG动画
  • Web Animations API 现代浏览器提供的原生JavaScript动画API，允许开发者以编程方式创建复杂的动画。
  • Motion 一个比较好用的动画库。
  • Motion Vue 建立在 Motion 之上的 Vue 动画库。
  • Tween.js 使用 JavaScript 中的一个简单的补间动画库，支持数字、对象的属性和 CSS 样式属性的赋值。
  • lottie 主要用法就是UI设计通过专业动画软件设计好之后，输出json文件，然后通过 Lottie 将这些动画无缝地集成到移动app和网页中

其他 ​

  • Z-Library 一个免费的电子书库，包含了大量的电子书资源。（需科学上网）
